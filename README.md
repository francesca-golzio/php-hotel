# 🏨 PHP Hotel

repo: `php-hotel`

## 📚 Assignment 

- **Print the List of Hotels** and their details
- **Add some Style** to the list of hotels

### 🎯 Bonus tasks
- **Add a form** to apply a **Parking Filter** to the list
- Add a **Rating Filter** to the form

#### 📌 Requirements
- The two **filters** must be able to **work simultaneously**
- If no filter is selected, **display all hotels**
